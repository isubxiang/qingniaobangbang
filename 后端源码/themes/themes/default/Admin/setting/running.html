注意：跑腿设置写在这里

跑腿基础费用： 每一次跑腿的基本费用，会员发布的时候收取，支持在线支付，新版如果筛选分类，在跑腿分类那调用费用
发布跑腿间隔时间： 秒为单位，建议填写60-900秒
会员发布跑腿小提示： 会员中心发布跑腿的时候的小提示，不超过30字！
请注意没有配置微信证书的千万不要开启，证书路径 根目录/cret/
开启取消订单原路退款： 证书路径/根目录/cret/需要上传3个文件
原路退款最大退款金额： 用户申请退款金额最大超过此数字后无法申请退款，不支持小数点，建议设置10元-100元只能填写数字，不要填写中文之类的
配送员跑腿结算费率
结算佣金： % 结算给跑腿的会员账户比如填写10%实际付款100元，给跑腿90元，网站扣除佣金100元
校园跑腿小程序配置
可同时抢几个订单： 可同时配送几个订单，建议设置1-3个，默认3个订单，不支持小数点
抢单间隔时间： 分钟 抢单一个后，再去抢单第二个需要间隔多少分钟才行，不填写默认1分钟，不支持小数点
执行价格开始时间： 比如：8:00主题英文小写的符号不要有空格不要乱写
执行价格结束时间： 比如：22:00主题英文小写的符号不要有空格不要乱写
运费起价说明： 比如填写，雨雪天等关键字
最低运费： 元 填写2-6不支持小数点
永许投递类型： 填写数字1-3不要乱填写
服务协议文章等设置
运费说明文章ID： 请去文章列表抄ID，请不要乱写
服务协议文章ID： 请去文章列表抄ID，请不要乱写
跑腿认证时候的配置
认证配送员是都开启身份证上传： 认证配送员是都开启身份证上传身份证，不开启就不上传
